Software for the Detection of Strand Bias in the NGS Data

In order to provide the scientific community better insight into strand bias and its manifestation in NGS data, SBAT, a command-line tool for analysis of strand bias in NGS was created at Faculty of Informatics at Masaryk University. This open-source tool is freely available at GitHub and also downloadable using pip. The tool accepts a .fasta dataset, analyses it based on frequencies of k-mers and produces visual and statistical output of analysis. SBAT was created as a part of a bachelor thesis, in which it was also used for an analysis of freely available datasets of Oxford Nanopore Technologies, Pacific Biosciences and Illumina. Based on the analysis, this webpage was created in order to make these results available even without the need to use SBAT. The interactive analysis offers multiple datasets to choose from and provides results in form of interactive Bokeh plots.
